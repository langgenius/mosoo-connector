# Release Manifests

This directory contains machine-readable metadata for Mosoo installer flows.

Use `install.example.json` as the routing template when configuring the public
`mosoo.ai` distribution endpoints. Replace example versions, sizes, and checksums
with values generated by CI before publishing a production manifest.

## Public Route Mapping

| Public URL | Backing source | Notes |
| --- | --- | --- |
| `https://install.mosoo.ai/install.sh` | `publish/installers/install.sh` | Serve as the stable Mosoo install script. Keep cache short, for example `max-age=300`. |
| `https://install.mosoo.ai/manifests/install.json` | generated production manifest from `publish/manifests/install.example.json` | Serve as JSON. Replace placeholder checksums before publishing. |
| `https://install.mosoo.ai/releases/latest/download/mosoo-darwin-arm64.tar.gz` | release artifact for `GOOS=darwin GOARCH=arm64` | Archive must contain an executable file named `mosoo`. |
| `https://install.mosoo.ai/releases/latest/download/mosoo-darwin-amd64.tar.gz` | release artifact for `GOOS=darwin GOARCH=amd64` | Archive must contain an executable file named `mosoo`. |
| `https://install.mosoo.ai/releases/latest/download/mosoo-linux-arm64.tar.gz` | release artifact for `GOOS=linux GOARCH=arm64` | Archive must contain an executable file named `mosoo`. |
| `https://install.mosoo.ai/releases/latest/download/mosoo-linux-amd64.tar.gz` | release artifact for `GOOS=linux GOARCH=amd64` | Archive must contain an executable file named `mosoo`. |
| `https://install.mosoo.ai/releases/latest/download/mosoo-skill.tar.gz` | archive generated from `publish/skills/mosoo` | Archive should unpack to `mosoo/SKILL.md`, with `references/` preserved. |
| `https://try.mosoo.ai` | Mosoo cloud API origin | The installer uses this as the default runtime target. |

## Installer Defaults

The current installer can already be pointed at these public artifacts without
code changes:

```sh
curl -fsSL https://install.mosoo.ai/install.sh | bash
```

Individual artifact URLs can also be overridden:

```sh
curl -fsSL https://install.mosoo.ai/install.sh | bash -s -- \
  --cli-url https://install.mosoo.ai/releases/latest/download/mosoo-darwin-arm64.tar.gz \
  --skill-url https://install.mosoo.ai/releases/latest/download/mosoo-skill.tar.gz
```
